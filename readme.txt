Plugin Invelity GLS connect je vytvorený pre obchodníkov na platforme Woocommerce ktorý potrebuju automaticky exportovat údaje o objednávkach do systému GLS online za ú?elom vytla?enia doru?ovacích lístkov.

== Description ==

Plugin Vám umožnuje jednoduchý prenos údajov o objednávkach z Wordpress adminu priamo do systému GLS connect pomocou exportovania/importovania súborov.

== Installation ==

Táto sekcia popisuje inštaláciu pluginu.

1. Stiahnite plugin a nahrajte ho priamo cez FTP (`/wp-content/plugins/invelity-gls-connect) alebo plugin stiahnite priamo z Wordpress repozitára.
2. Aktivujte plugin cez 'Plugins' obrazovku vo WordPress.
3. V hlavnom menu (ľavý sidebar) uvidíte položku "Invelity plugins" a jej pod-položku "Invelity GLS connect".
4. Vpíšte všetky potrebné údaje vrátane údajov ktoré ste dostali priamo od služby GLS.
5. Po správnom nastavení pluginu môžete pristúpiť k exportovaniu údajov o objednávkach do GLS.
6. Vo výpise objednávok zaškrtnite objednávky ktoré chcete exportovať do GLS. Z drop-down zvoľte možnosť "Export GLS connect"
7. Stiahnite vygenerovaný súbor a importujte ho do GLS connect
